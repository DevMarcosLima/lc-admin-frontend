# LC Admin Frontend

React admin panel for Legacy Cards.

## Quick start

```bash
npm install
cp .env.example .env
npm run dev
```

## URL

- `http://localhost:5174`

## Required

- Backend admin API running em http://localhost:8001
- Login admin via e-mail/senha
- Se `ADMIN_AUTH_2FA_ENABLED=true`, informar codigo do Google Authenticator
